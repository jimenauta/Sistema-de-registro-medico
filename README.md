# 🏥 Sistema de Registro Médico

Aplicación web para registrar, buscar y listar pacientes en un entorno hospitalario.

## 📌 Descripción del problema

Los hospitales y clínicas necesitan digitalizar el registro de sus pacientes para evitar pérdida de información, agilizar las búsquedas por número de cédula y tener un listado organizado del historial médico de cada persona atendida.

## 💡 Solución

Se desarrolló una aplicación web con frontend y backend conectados mediante una API REST que permite:
- **Registrar** un nuevo paciente con nombre, cédula, edad, email y síntomas
- **Buscar** un paciente existente por su número de cédula
- **Cargar** el listado completo de todos los pacientes registrados en el sistema

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)
- Python
- Flask (backend y API REST)
- JSON (almacenamiento de datos)
